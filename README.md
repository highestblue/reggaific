## Author

## Description

Music player webapp for reggae lovers. Best reggae handpicked for you!

## Goal

To spoonfeed jewels of reggae music to the users

## Target Audience

- Reggae fanatics
- Music lovers
- Multi-device users

## Features

- Responsive
- Playlist
- Search by title or artist name
- Play & pause
- Skip to previous & next
- Loop
- Volume control
- Change playback position
- Timeline feedback
